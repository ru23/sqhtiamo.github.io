## 基本操作
```
control + a: 到行首
control + e: 行末
control + f/b: 前进后退，相当于左右方向键，但是显然比移开手按方向键更快
control + p: 上一条命令，相当于方向键上
control + r: 搜索命令历史，这个大家都应该很熟悉了
control + d: 删除当前字符
control + h: 删除之前的字符
control + w: 删除光标前的单词
control + k: 删除到文本末尾
control + t: 交换光标处文本
control + u: 删除一行
 + —/+/0: 调整字体大小
⌘ + r:清屏，其实是滚到新的一屏，并没有清空。ctrl + l 也可以做到
```
